# 广业材价部署说明（极简版）

## 1. 执行什么脚本

### 1.1 数据库全量同步（必做）
在项目根目录执行：

```bash
cp .deploy.env.example .deploy.env
# 编辑 .deploy.env，填写数据库信息
# 重点：DEPLOY_CONFIRM=YES、ENABLE_PACKAGE=0
bash scripts/deploy_full_sync.sh ./.deploy.env
```

脚本作用：
- 先备份线上库
- 把本地库全量导入线上（表 + 字段 + 数据）
- 校验表清单一致、字段结构一致、逐表行数一致

### 1.2 代码部署（你单独给运维代码包时）
按你提供的代码包部署即可，本脚本不打包代码（`ENABLE_PACKAGE=0`）。

## 2. 注意什么事情

- 脚本会执行“重建线上库 + 全量导入”，必须在维护窗口执行。
- 目标库信息必须确认无误，避免导错库。
- 先确认线上账号具备建库/删库/导入权限。
- 任何密钥不要写入仓库，走服务器环境变量或运维密钥系统。
- 出现异常时，用 `runtime/deploy/prod_backup_*.sql` 回滚。
