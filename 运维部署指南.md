## 一、数据库操作（按顺序执行）

### 1. 执行菜单补丁
```bash
mysql -u [用户名] -p [数据库名] < gz-springboot-base-dev/sql/menu_patch.sql
```
**作用**: 创建"供应商查询"和"材料标准"菜单

### 2. 执行供应商表初始化
```bash
mysql -u [用户名] -p [数据库名] < gz-springboot-base-dev/sql/supplier_init.sql
```
**作用**: 创建 `supplier_info` 表及示例数据

---

## 二、代码部署

### 1. 后端部署
```bash
cd gz-springboot-base-dev
git pull origin main
mvn clean package -DskipTests
# 重启服务
```

### 2. 前端部署
```bash
cd gz-vue-base-dev
git pull origin main
pnpm install
pnpm build
# 部署到 Web 服务器
```
